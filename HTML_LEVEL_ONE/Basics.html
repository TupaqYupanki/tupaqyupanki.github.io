    HTML Level One - Part Three

    Lists

    For ordered list use <ol> for list and <li> for items in list:

    1. First item
    2. Second item

    For unordered list use <ul> for list and <li> for items in list:

    • First item
    • Second item

    Also we can did nested lists:

    1. First item
    2. Second item
      1. First nested item
      2. Second nested item
        • Another nested item

    HTML Level One - Part Four

    Divs and Spans

    Tags <div> and <span> group a part of text to apply some CSS styles on it

    <div> can group paragraphs and <span> inline part of text.

    Both tags have some attributes <div class="attributes">

    For insert some files at the page we can use <img src="image.png" alt="">

    src part is for image source, alt - for comment if image is missing
