The Bootstrap is popular open source framework for frontend.
It can be reached by the link - https://getbootstrap.com/
Here is a few ways to use the bootstrap:
  • Download the framework at your local server
  • Link it by CDN (Content Delivery Network). Actual link you can find at https://getbootstrap.com/docs/5.1/getting-started/download/
