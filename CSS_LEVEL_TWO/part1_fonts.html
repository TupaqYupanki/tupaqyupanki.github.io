For changing Fonts attribute use element font.
Propterty Font has many parameters like "size", "family", "style", etc.

Font-size have many value like "px", "%", "em". Two last is relative value.
em - is current font size. So we can set value relatively like 1.5em
So if we set the body font-size as 20px, p with font-size 1.5em will be equal 30px.

text-align - the property that set horizontal alignment of a text.

We can use not only buil-in OS fonts but use some remotely located fonts by linking it to ytour page.
For example:
<link href="https://fonts.googleapis.com/css2?family=Barrio&display=swap" rel="stylesheet"> code adding remotely font family named "Barrio"
To use it you need to apply it in CSS with font-family: 'Barrio', cursive; parameter.
