Selecting with jQuery

You can use the $ sign to select elements from the DOM! It will feel a lot like selecting using querySelectorAll().

For example:

  • Selecting all a tags:
    • $('a')
  • Selecting all elements from the class "container"
    • $(".container")
  • Selecting all elements with id "special"
    • $("#special")

jQuery events

.click() example
$('h1').click(function(){
console.log('Ther was a click!');
})
another example:
$('h2').click(function(){
$(this).text("I was changed")
})

In this example we use function eq() of jQuery that takes numbers as argument and goes to the element in list of elements. Than we use function keypress() that take as argument some action

$('input').eq(0).keypress(function(){
$('h3').toggleClass('turnBlue')
})

In the next example we using special keyword "event" that contains list of attrimutes with information about actions on the page

$('input').eq(0).keypress(function(event){
console.log(event);
})

event.which contain number code of key that was pressed. For example number code '13' is 'enter' keyboard

$('input').eq(0).keypress(function(event){
if (event.which === 13) {
$('h3').toggleClass('turnRed')
}
})

Another option of interaction is method on()

$('h1').on('dblclick',function(){
$(this).toggleClass('turnBLue');
})

Example with 'mouseenter' for 'li'

$('li').on('mouseenter',function(){
$(this).toggleClass('turnRed');
})

Animation in jQuery https://api.jquery.com/animate/

$('input').eq(1).on('click',function(){
$('.container').fadeOut(3000) //3000 - numer of ms after fadeout
})
