Selecting with jQuery

You can use the $ sign to select elements from the DOM! It will feel a lot like selecting using querySelectorAll().

For example:

  • Selecting all a tags:
    • $('a')
  • Selecting all elements from the class "container"
    • $(".container")
  • Selecting all elements with id "special"
    • $("#special")

To choose some element using jQuery use simbol $
For example:

> $('h1') < [h1]

To change some paramaters of element choose method and change parameters:

var x = $('h1')
x.css('color','red')
x.css('bacground','blue')

We can create JS object with CSS parameters and then apply it to element:

var newCSS = {
'color':'white',
'background':'green',
'border':'2px solid red'
}

var y = $('li')
y.css(newCSS)

To choose particular element from list use method "eq":

var y = $('li')
y.eq(0).css('color','orange')
y.eq(-1).css('background','yellow')
y.eq(-1).css('color', 'black')

To grab text of element use method ".text"

> $('h1').text('NEW TEXT')
> $('h1').html()
< "NEW TEXT"

With jQuery we can change attribute of element:

$('input').eq(1).attr('type','chekbox')    //choose second element of 'input' and change attribute 'type' for button
$('input').eq(0).val('new value')    //choose the first element of 'input and change 'value'

If we want to add or remove predifined class of CSS we need method '.addClass' or '.removeClass' of jQuery:

> $('h2').addClass('turnRed')
< [h1.turnRed]
> $('h2').removeClass('turnRed')
< [h1]

Another option of adding or removing class method '.toggleClass'

> $('h2').toggleClass('turnRed')
< [h2.turnRed]
> $('h2').toggleClass('turnRed')
< [h2]
