Class is the selector of some elements like <div>, <span>, <p>, etc.
Class can be attached to multiple elements.
In CSS it defines with dot before class name.
Example:

<div class="className"> - HTML code
.className{color: red;} - CSS code

id is the unique selector. it attached to single element.
ids defines with # symbol before id's name
Example:

<span id="idName"> - HTML code
#idName{color: red;} - CSS code

Another selector is *. It's selector for every elements in the code.
It's defines in CSS code like any parameters and override all the elements exept ids
