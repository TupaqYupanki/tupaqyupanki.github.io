Tag <div> specifies the part of text like object to wick we can apply some CSS parameters.

For example we can add background-color to <body> or <div>

Another parameter of CSS is border

border-width determines width of border with named value: "thin", "medium" or "thick" or custom value in "px"

For background we can use some image from remote server using remote link

For example:

body{
background: url(https://png.pngtree.com/thumb_back/fw800/background/20190428/pngtree-bokeh-light-in-yellow-blue-gradient-background-image_104146.jpg)
background-repeat: no-repeat;
background-position: center;
background-size: cover;
height: 100%;
}
