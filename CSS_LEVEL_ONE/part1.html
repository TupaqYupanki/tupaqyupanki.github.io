To link CSS file to HTML we need to add string: <link rel="stylesheet" href="part1_master.css">

For comments in CSS use /* some comments */

Basic format for CSS is:

selected tag{
property: value;
}

For example:

h1{
color: blue;
}

Color can be like word - "green", "red", etc. or like RGB code - rgb(10,150,10) or hex - #4286f4 or RGBA - rgba(10,150,10,0.5), where Alpha is for opacity of color. 0 - fully transparent, 1 - fully opaque
