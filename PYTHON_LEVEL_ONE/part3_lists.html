To create a List you need to create some item with []
>>> mylist = [1,2,3]
>>> mylist
[1,2,3]

The first method of lists is len() method. It gives us the number of elements in the list.
>>> mylist = [1,2,3]
>>> len(mylist)
3

We can get the element from the list similarly like in strings with mylist[x]
>>> mylist = ['a','b','c']
>>> mylist[1]
a

Unlike a string list's items are mutable
>>> mylist = [1,2,3]
>>> mylist[0] = 'new item'
>>> mylist
['new item',2,3]

To append new item to the list use method .append()
>>> mylist = ['a','b','c']
>>> mylist.append('new item')
>>> mylist
['a','b','c','new item']

If we want to add items from another list use method .extend()
>>> mylist = ['a','b','c']
>>> mysecondlist = ['d','e','f']
>>> mylist.extend(mysecondlist)
>>> mylist
['a','b','c','d','e','f']

One of the weay to get item from list is .pop() method. It get the last item from the list.
>>> mylist = ['a','b','c','d']
>>> x = mylist.pop()
>>> mylist
['a','b','c']
>>> x
d

If we want get item by the number use .pop(x), where 'x' is number of item.
