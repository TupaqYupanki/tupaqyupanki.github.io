1. Install terminal for Atom:
    Go to File -> Settings -> Install, find package "platformio-ide-terminal" and install it.
    Go to settings of package to "Shell override" option and change it on path to cmd.exe (C:\WINDOWS\System32\cmd.exe)
  2. Then find "autocomplete-python" package and install it
  3. Finaly, find and install "atom-django" package.

Go to main menu
