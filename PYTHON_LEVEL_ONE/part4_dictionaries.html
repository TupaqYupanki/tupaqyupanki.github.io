To create dictionarie object use {}
>>> myDict = {'key1':'value1', 'key2':'value2'}
>>> myDict
{'key1':'value1', 'key2':'value2'}
>>> myDict['key1']
value1

>>> my_stuff = {'key1':123, 'key2':'value2', 'key3':{'123':[1,2,'grabMe']}}
>>> my_stuff
{'key1': 123, 'key2': 'value2', 'key3': {'123': [1, 2, 'grabMe']}}
>>> my_stuff['key3']['123'][2]
'grabMe'

To add or change value of key use this:
>>> my_stuff = {'key1':123, 'key2':'value2', 'key3':{'123':[1,2,'grabMe']}}
>>> my_stuff['key2'] = 'egg'
>>> my_stuff['key2']
'egg'

>>> my_stuff['key4'] = 'pasta'
>>> my_stuff
{'key1': 123, 'key2': 'egg', 'key3': {'123': [1, 2, 'grabMe']}, 'key4': 'pasta'}
