To go to specific character of string use mystring[x], where 'x' is serial number of character
>>> mystring = 'abcdefg'
>>> print(mystring[3])
d

Slicing

Here are 3 parameters of slicing: the begining of the slice, the end and step size.
>>> mystring = 'abcdef'
>>> print(mystring[2:])
cdef

>>> print(mystring[:3])
abc

>>> print(mystring[2:5])
cde

>>> print(mystring[::2])
ace

Python strings are immutable!
So you can't change some character in a string.

Strings have a lot of methods. Here is the most useful one .split()
>>> mystring = 'Hello World!'
>>> print(mystring.split('e'))
['H', 'llo World!']

We can insert another string to string with method .format()
>>> x = 'Item One: {} Item Two: {}'.format('dog', 'cat')
>>> print(x)
Item One: dog Item Two: cat

>>> x = 'Item One: {x} Item Two: {y}'.format(x='dog', y='cat')
>>> print(x)
Item One: dog Item Two: cat

>>> x = 'Item One: {y} Item Two: {x}'.format(x='dog', y='cat')
>>> print(x)
Item One: cat Item Two: dog
