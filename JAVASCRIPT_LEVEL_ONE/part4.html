Booleans

Comparison operation

Greater Than

3 > 2;
2 > 3;

Less Than

1 < 2;

Greater than or equal to

1 <= 3;

Equality

2 == 2;
"name" == "name"

Inequality

2 != 3;

If tried to compare two elements with different types JS will compare it like one type. But if we need to compare elements considering the type we have to add another equal operator
For example:

> 3 == "3";
< true

> 3 === "3";
< false

Logical operators

AND is written with &&

Need both conditions to be true to return true

> 1 == 1 && 2 == 2;
< true

OR is written with ||

Need only one condition to be treu to return true

> 3 == 1 || 2 == 2;
< true

NOT is written with !

> (3 == 1);
< false

> !(3 == 1);
< true
