Control Flow

IF statement

This statement compare conditions and if result true execute some code

if (x > 0){
z = x**2;
}

IF ELSE statement

This statement compare conditions and if result true execute some code else execute some other

if (x > 0){
z = x**2;
}else{
z = z+x;
}

IF, ELSE IF, ELSE statement

if (x > 0){
z = x**2;
}else if(z != 0){
z = z+x;
}
