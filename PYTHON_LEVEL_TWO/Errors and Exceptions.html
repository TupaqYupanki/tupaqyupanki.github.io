We can use these keywords:
  • Try
  • Except
  • Finally
Basic syntax:

try:
    You do your operations here
except Exception1:
    If there is Exception1, then execute this block
except Exception2:
    If there is Exception2, then execute this block
else:
    If ther is no exception then execute this block

try:
    f = open('simple.txt','w')
    f.write('Test write to simple text')
except IOError:
    print('ERROR: COULD NOT FIND FILE OR READ DATA')
else:     print('SUCCESS')
    f.close()
print('hello world')
In this example if something went wrong and our program can't open file or write data to it the program wouldn't crash and continue performing the code
