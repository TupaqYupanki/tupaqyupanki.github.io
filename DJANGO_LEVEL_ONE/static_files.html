To add some media files to your page you can use static files.
First, you need to create new directory inside your project called static
Then need to add this directory path to the project's settings.py file. We will aslo add a STATIC_URL variable
After that we create images directory inside of static and can place our image files here.
Now we can go to 127.0.0.1:8000/static/images/pict.jpg and check that it works.
