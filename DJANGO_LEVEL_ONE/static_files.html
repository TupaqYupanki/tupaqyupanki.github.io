To add some media files to your page you can use static files.
First, you need to create new directory inside your project called static
Then need to add this directory path to the project's settings.py file:
STATIC_DIR = os.path.join(BASE_DIR,"static")

We will also add a STATIC_URL variable:
STATICFILES_DIRS = [
    STATIC_DIR,
]

 After that we create images directory inside of static and can place our image files here.
Now we can go to 127.0.0.1:8000/static/images/pict.jpg and check that it works.

To add this media file to HTML we need to add specific tags at the top (under <html> tag):
{% load staticfiles %}

Then we want to insert the image with an HTML <img src= > style tag using:
<img src={% static "images/pic.jpg" %}/>

Also we can add CSS file in static field:
Create new directory named css, add in this directory some css file, for example "mystyle.css".
Than we need to add in our html file link to it:
Inside <head> tag add <link rel="stylesheet" href= '{% static "css/mystyle.css"%}'>
