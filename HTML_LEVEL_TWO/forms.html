Forms

Form is used to collect user input. The user input is most often sent to a server for processing.

For creating form using element <form>.

<form> have some attribute like "action=" and "method=".

Attribute "action=" defines the action to be performed when the form is submitted. Usually, the form data is sent to a file on the server when the user clicks on the submit button.

The "method" attribute specifies the HTTP method to be used when submitting the form data. The form-data can be sent as URL variables (with method="get") or as HTTP post transaction (with method="post").

Input

For collecting user data using element <input>

<Input> tag have some attributes: type, name and value, etc.

Attribute "type" determine the type of information. It have several value: text, color, email, password, submit, etc.

Attribute "name" specifies the name of input element

Attribute "value" specifies the value of input element

Attribute "placeholder" the same as value but dissapears when tried to fill the form

Attribute "required" makes the field requiring to fill



Label

<label> using for adding some description for <input> element.

it can be lnked with attribute "id" in <input> element

For example:

<form>

   <label for="fname">First name:</label>

   <input type="text" id="fname" name="fname"><br>

   <label for="lname">Last name:</label>

   <input type="text" id="lname" name="lname">

</form>
