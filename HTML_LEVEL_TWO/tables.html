Tables

For table use element <table>. This element declare start and end of the table

Element <thead> is for header of the table that have bold text

Tags <th> is part of the header column of the table

Tag <tr> - table row, declare the new row of cells

Tags <td> is for each element of table cells

So the construction of the table looks like:

<table>

   <thead>

      <th>First header of column</th>

      <th>Second header of column</th>

      <th>Third header of column</th>

   </thead>

   <tr>

      <td>First cell in row</td>

      <td>Second cell in row</td>

      <td>Third cell in row</td>

   </tr>

</table>
