In this example we show how interact with some elements using JS using DOM

At the first we grap the element h1 using DOM method document.querySelector()

var header = document.querySelector("h1")

Next we create some function that will randomly generate hash color

function getRandomColor(){
var letters = "0123456789ABCDEF";
var color = '#';
for (var i = 0; i < 6; i++) {
color += letters[Math.floor(Math.random()*16)];
}
return color
}

The next we create function that will change property of tag h1 using DOM method .style.color

function changeHeaderColor(){
colorInput = getRandomColor()
header.style.color = colorInput;
}

The last thing is apply changes over interval in ms

setInterval("changeHeaderColor()",500);
