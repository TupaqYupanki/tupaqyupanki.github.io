Document Object Model

DOM presents the HTML document as a JavaScript object.
It allow us to grap properties of the web page and edit the values to create changes that appear to the user on the frontend
So you can use JS to manipulate what the users sees on the web page

You can show DOM of any html page typed in console console.dir(document)


Here are some important document attributes:

  • document.URL - This is the actual URL of the website
  • document.body - This is everything inside of the body
  • document.head - This is everything in the head of the page
  • documetn.links - These are all the links on the page

There are many methods for grabbing elements from the DOM:

  • document.getElementById() - Returns the element with the id
  • document.getElementByClassName() - Returns list of all elements belonging to a class
  • document.getElementsByTagName() - Returns list of all elements with the tag
  • document.querySelector() - Returns the first object matching the CSS style selector
  • document.querySelectorAll () - Returns all objects matchin the CSS style selector
