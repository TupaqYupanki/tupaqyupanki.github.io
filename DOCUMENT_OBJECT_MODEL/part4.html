Interact with DOM on events

We can interact with elemtns of DOM with JavaScript throug Event Listener.
The JavaScript will be "lisening" for an event to occur and then execute a function whtn it happens.

For example:
var myvariable.querySelector('h1')
myvariable.addEventListner('click', myFunction(){
})

There are many possible events:

  • Clicks
  • Hovers
  • Double Clicks
  • Drags
  • ...
More events

HOVER EXAMPLE

var headOne = document.querySelector('#one')

headOne.addEventListener('mouseover', function(){
headOne.textContent = "Mouse Currently Over";
headOne.style.color = 'red';
})

headOne.addEventListener('mouseout', function(){
headOne.textContent = 'Hover example';
headOne.style.color = 'black';
})

CLICK EXAMPLE

var headTwo = document.querySelector('#two')

headTwo.addEventListener('click', function(){
headTwo.textContent = 'CLICKED ON'
headTwo.style.color = 'blue'
}
)

DOUBLE CLICK EXAMPLE

var headThree = document.querySelector('#three')

headThre.addEventListener('dblclick', function(){
headThree.textContent = 'I WAS DOUBLE CLICKED!'
headThree.style.color = 'green'
}
)
