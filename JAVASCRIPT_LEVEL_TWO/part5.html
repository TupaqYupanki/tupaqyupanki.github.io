Object

In JS Object is a dictionary. Unlike the arrow it didn't have any order inside. You can access to values through their key.
The general syntax for object is:

{key1: 'value one', key2: 'value two'}

Useful link - https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/Object
Examples:

var carInfo = {make:'Toyota', year:1990, model:'Camry'}

Keep in mind that "key" didn't have quotes when you defines it in object. But when you need to access to value with the key you have to add quotes in your request.

> carInfo
< {make: 'Toyota', year: 1990, model: 'Camry'}

> carInfo['year']
< 1990

var myNewO = {a:'hello', b:[1,2,3], c:{inside:['a','b']}};

> myNewO
< {a: 'hello', b: Array(3), c: Object}

> myNewO['a']
< 'hello'
> myNewO['b']
< (3) [1, 2, 3]
> myNewO['b'][0]
< 1
> myNewO['c']['inside'][1]
< 'b'

To change some value just use objectName['key1'] = 'newValue'

carInfo['year'] = 2006;

To show at browser the Object you can use console.dir(carInfo)

for (i in carInfo){
console.log(i);
console.log(carInfo[i]);
}

Object methods

Object methods are functions that are inside of an Object

var carInfo = {
make:'Toyota',
year:1990,
model:'Camry',
carAlert: function(){alert("We've got a car here!")} }

To call this method use syntax carInfo.carAlert()
Another way to call the function is using keyword this.
This keyword set to the object the method is called on. So this is acting as a reference to the actual object.

var myObj = {
protp: 37,
reportProt: function() {
return this.prop;
}
}

var simple = {
prop: "Hello",
myMethod: function(){
console.log("The myMethod was called");
}
}

To calling function we need to add () at the end of function

> simple.myMethod
< ƒunction (){
console.log("The myMethod was called");
}

> simple.myMethod()
< The myMethod was called
