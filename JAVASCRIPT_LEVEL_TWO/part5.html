Object

In JS Object is a dictionary. Unlike the arrow it didn't have any order inside. You can access to values through their key.
The general syntax for object is:

{key1: 'value one', key2: 'value two'}

Useful link - https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/Object
Examples:

var carInfo = {make:'Toyota', year:1990, model:'Camry'}

Keep in mind that "key" didn't have quotes when you defines it in object. But when you need to access to value with the key you have to add quotes in your request.

> carInfo
< {make: 'Toyota', year: 1990, model: 'Camry'}

> carInfo['year']
< 1990

var myNewO = {a:'hello', b:[1,2,3], c:{inside:['a','b']}};

> myNewO
< {a: 'hello', b: Array(3), c: Object}

> myNewO['a']
< 'hello'

To change some value just use objectName['key1'] = 'newValue'

carInfo['year'] = 2006;
