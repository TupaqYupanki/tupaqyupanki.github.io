Arrays

Arrays allow us to store various basic data types in a sequence and get them by index while it need

The general syntax for array is:

var arrayName = ['var1', 'var2', 'var3']

> arrayName[0]
< 'var1'

We can change value of array's item:

arrayName[2] = "var4";

> arrayName
< ['var1', 'var2', 'var4']

Remember that we can change value of array but not string. Arrays are mutable, string - not.

var stringName = 'Hello World!';
> stringName[0]
< 'H'

stringName[0] = 'Y';
> stringName[0]
< 'H'

To receive length of array use method arrayName.length

> arrayName.length
< 3

Other methods you can find at https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/Array#

To add or remove items use .push() and .pop() methods

var myArr = ['one', 'two', 'three']
var lastItem = myArr.pop()

> lastItem
< 'three'
> myArr
< ['one', 'two']

myArr.push('new_item')
> myArr
< ['one', 'two', 'new_item']

We can did matrix with nested arrays. Arrays inside the array.

var matrix = [[1,2,3], [4,5,6],[7,8,9]]
> matrix
< [Array(3), Array(3), Array(3)]

> matrix[1][0]
< 4

Array iteration

var arr = ['A', 'B', 'C'];
for (var i = 0; i < arr.length; i++) {
console.log(arr[i]);
}

for (letter of arr){
console.log(letter);
}

We can use method .forEach instead for

for (letter of arr){
alert(letter)}

arr.forEach(alert)

More examples:

function addAwesome(name){
console.log(name+' is awesome!');
}

var topics = ['python', 'django', 'science']

> topics.forEach(addAwesome)
< python is awesome!
< django is awesome!
< science is awesome!
