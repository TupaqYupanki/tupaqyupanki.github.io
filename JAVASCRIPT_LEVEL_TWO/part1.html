FUNCTIONS

Functions allows us reuse part of code

The general syntax for a JS function is:

function name(parameter1, parameter2){
//Code to be executed
}

Examples:
function hello(){
console.log("hello world!");
}

function helloYou(name){
console.log("hello "+name);
}

function addNum(num1, num2){
console.log(num1+num2);
}

We can predifines the parameter of function

function helloYou(name='Frankie'){ console.log("hello "+name); }

To return result of the function we can use operate "return"

function formal(name='Sam', title='Sir'){ return(title+' '+name); }

All defined variables inside function are limited of this function and doesn't exist outside the function

So we have to scopes - global (outside the function) and local (inside the function). We can work and change global variables in our function but it didn't changes globaly.

var x = "global x"
function fun(x){
x = "reassigned x"
console.log('this is reassigned x inside function: '+x);
}
console.log('this is global: '+x);
fun();
console.log('global x is uncnhanged: '+x);

< this is global: global x
< this is reassigned x inside function: reassigned x
< global x is uncnhanged: global x
